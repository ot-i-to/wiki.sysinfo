При наличии нескольких одинаковых сетевых карт, чтобы гарантировать, что имя интерфейса в виде eth0, eth1, всегда будут закреплены за определенным Ethernet портом. Это можно огранизовать привязав имя интерфейса к MAC адресу.

В /etc/mactab добавляем:   eth0 00:23:B7:89:39:E2   eth1 00:08:A3:20:F8:29   eth2 00:50:BA:29:B7:32 Для Debian GNU/Linux создаем скрипт /etc/network/if-pre-up.d/nameif   #!/bin/sh   PATH=/sbin   nameif -c /etc/mactab Для других Linux дистрибутивов нужно на начальном этапе инициализаци выполнить:   /sbin/nameif eth0 00:23:B7:89:39:E2   /sbin/nameif eth1 00:08:A3:20:F8:29   /sbin/nameif eth2 00:50:BA:29:B7:32
