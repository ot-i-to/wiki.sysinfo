Давайте опишим как настроить GRUB для загрузки любого IMG образа лежащего на локальном диске линукса ( в нашнм случае Debian 12 ). Возмем для примера образ FreeDOS созданный посредсвам DD команды с USB-Flash. 

1) Сперва надо установить приложением grub-imageboot:
sudo apt install grub-imageboot

2) Потом проверим наличие файла /etc/grub.d/60_grub-imageboot используемого для автоматизации настроек GRUB.

3) Создадим директорию /boot/images в которой по умолчанию ищит IMG образы вышеупомянутый скрипт автоматической настройки:
sudo mkdir -p  /boot/images
