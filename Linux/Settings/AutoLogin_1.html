Включаем автоматический вход в графический интерфейс Linux Mint 

Для этого надо откорректировать или создать файл в случае его отсутсвия /etc/lightdm/lightdm.conf со следующим содержанием:

[Seat:*]
autologin-guest=false
autologin-enable=true
autologin-user=USERNAME
autologin-user-timeout=0
allow-guest=false

Где USERNAME — это ваше имя пользователя, которое должно автоматически входить в систему, а строка autologin-enable=true включает автоматический вход. 

Проверено на Linux Mint 20.3 MATE :)
