Когда есть необходимость полностью убить все сессии конкретного пользователя (username) завершив их принудительно. 

То можно воспользоваться следующей командой через терминал:

sudo pkill -9 -u username

Но иногда требуется принудительно завершить сессию только одного терминала. 

Тогда подойдет этаже команда но с изменениями:

sudo pkill -9 -е TTY-name

где TTY-name в виде “pts/2” можно узнать из команды “w” :
 

# w
10:36:55 up 177 days, 18:26,  2 users,  load average: 0,37, 0,52, 0,58
USER     TTY      FROM              LOGIN@   IDLE   JCPU   PCPU WHAT
user1   pts/0    10.50.4.59       Mon09   43:03m  0.14s  0.02s sshd: user1 [priv]
user2   pts/2    10.50.4.59       23:22    0.00s  0.01s  0.01s sshd: user2 [priv]
