В этой статье описана пошаговая инструкция, как откатить прошивку MikroTik RouterOS к старой версии не используясь Netinstall даже если вы желаете с 7 версии вернутся к 6 версии.

  • Откройте сайт www.mikrotik.com и перейдите в меню Software - Downloads и скачиваем старую версию прошивки.

Внимание! Производитель MikroTik выпускает оборудование на разных процессорах с разной архитектурой, поэтому для разных устройств нужна разная прошивка. Первым делом определим, открыв в браузере WebFig интерфейсе меню System, раздел Resources в параметре Architecture Name смотрим архитектуру.

А на сайте в таблице, первом столбце указаны архитектуры MIPSBE, SMIPS, TILE, PPC, ARM, X86, MMIPS. Справа указаны серии устройств, которые относятся к данной архитектуре.

Мы будем устанавливать старую прошивку на роутер hAP ac lite tower. Ищем среди серий устройств, похожее название. Находим серию hAP ac и видим слева, что она относится к архитектуре MIPSBE. Значит для нашего роутера нужен будет файл прошивки, у которого в названии будет присутствовать MIPSBE, например, routeros-mipsbe-6.43.14.npk

Теперь переходим на сайте в меню Download archive.

Ниже есть две ветки с прошивками:

  • Long-term release tree (самая стабильная)
  • Stable release tree (стабильная + новые функции).

Выберем прошивку из ветки Long-term release tree с версией Release 6.43.14. Перед вами откроется список прошивок для разной архитектуры. Для нашего роутера с архитектурой MIPSBE ищем файл routeros-mipsbe-6.43.14.npk и скачиваем его на компьютер.

Загрузим старую прошивку в роутер MikroTik и установим ее:

  • В программе WinBox или через браузер в WebFig интерфейсе откройте меню Files.
  • Не переходя из корня в другие папки, через кнопку Upload  закачайте ранее скаченный файл прошивки с расширением .npk.
  • После загрузки, файл прошивки отобразится в окне File List.
  • Далее откройте меню System - Packages и нажмите кнопку Downgrade. Подтвердите установку старой версии RouterOS, нажав Yes. Роутер перезагрузится и установит старую прошивку.

После перезагрузки, как видим, операционная система откатилась до версии 6.43.14.
