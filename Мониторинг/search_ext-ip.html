Свой внешний IP адрес можно узнать с помощью одной из команд:

wget -qO- icanhazip.com
wget -qO- ip.mysokol.ru
wget -qO- ifconfig.me/ip
wget -qO- http://formyip.com/ | awk '/The/{print $5}'
wget -qO- http://checkip.dyndns.com/ | awk '{print $6}' | sed 's/<.*>//' 
wget -qO- ident.me 
wget -qO- eth0.me
wget -qO- ipecho.net/plain
wget -qO- ipecho.net
wget -qO- myip.gelma.net

curl -s checkip.dyndns.org | grep -Eo '([0-9]{1,3}\.){3}[0-9]{1,3}'
curl ipinfo.io/ip
curl v4.ident.me
curl ident.me
curl v4.ifconfig.co
curl v6.ifconfig.co
curl ifconfig.me
curl 2ip.ru
curl internet-lab.ru/ip
curl icanhazip.com
curl whatsmyip.akamai.com
curl tnx.nl/ip
curl ipecho.net/plain
curl ip.appspot.com
curl api.ipify.org

lynx --source http://formyip.com/ | awk '/The/{print $5}'

С помощью nslookup:

nslookup myip.opendns.com. resolver1.opendns.com

Чтобы не набирать длинную команду каждый раз, можете создать алиас:

alias extip="curl 2ip.ru"

Чтобы сохранить alias, нужно прописать выполнение команд alias в файл . bashrc пользователя. Порсмотреть список alias командой alias -p. Удалить по имени, unalias имя_alias. Удаление всех alias, unalias -a.
