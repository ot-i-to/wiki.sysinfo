Всегда ли проходить через DNS-сервер перед отправкой строки поиска из одного слова в поисковую систему. ⇡

GoToIntranetSiteForSingleWordEntryInAddressBar
Принудительно использовать прямую навигацию по сайту в интрасети вместо поиска при вводе отдельных слов в адресной строке.

Если сайт существует, произойдет переход на него. Если интрасеть отвечает кодом 404, на странице отобразится ошибка 404. Если интрасеть не отвечает, браузер попытается выполнить поиск.

Второй результат в строке URL-адреса будет результатом поиска, позволяющим пользователям выполнять поиск в Интернете точно так, как он был введен.

Если вместо этого вы хотите включить возможность отображать ваш домен как действительный URL-адрес и запретить браузеру когда-либо выполнять поиск по этому термину с использованием первого соответствующего ему результата, добавьте преф browser.fixup.domainwhitelist.ВАШ_ДОМЕН (где ВАШ_ДОМЕН — это имя домена, который вы хотите добавить) и установите для параметра значение true. Затем в строке URL будет предложено ВАШ_ДОМЕН, когда пользователь полностью введет ВАШ_ДОМЕН. Если пользователь попытается загрузить этот домен, но он не загрузится, отобразится страница с ошибкой «Невозможно подключиться».

Вы также можете внести в белый список суффикс домена, который не входит в список общедоступных суффиксов, добавив преф browser.fixup.domainsuffixwhitelist.ВАШ_ДОМЕН_СУФФИКС со значением true.

Кроме того, если вы хотите, чтобы пользователи видели подсказку «Вы хотели перейти к «ВАШ_ДОМЕН»» под строкой URL-адреса, если они попадают на страницу результатов поиска, а не на домен интрасети, который предоставляет ответ, установите преф browser.urlbar.dnsResolveSingleWordsAfterSearch равным 1. Включение этого параметра приведет к тому, что браузер будет выполнять проверку DNS после каждого поиска отдельного слова. Если браузер получает ответ из интрасети, пользователю будет предложено перейти на сайт ВАШ_ДОМЕН. Если пользователь нажмет кнопку «yes», для параметра browser.fixup.domainwhitelist.ВАШ_ДОМЕН будет установлено значение true.
