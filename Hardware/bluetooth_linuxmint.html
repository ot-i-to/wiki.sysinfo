Для устранения проблем с Bluetooth в линух Linux “br-connection-unknown”.

1) проверяем в терминале что-бы безопасная загрузка была отключена :

~ $ mokutil --sb
SecureBoot disabled
Platform is in Setup Mode

 … SecureBoot жолжен имеь значение disabled

2) Затем :

sudo apt install git dkms
git clone https://github.com/jeremyb31/bluetooth-6.5.git
sudo dkms add ./bluetooth-6.5
sudo dkms install btusb/4.1

… учти что значение 6.5 указывает / соответствует номеру версии ветки ядра … я проверял на версиях 6.5 и 6.8 

3) Далее, обязательно перегрузите компьютер … например командой Reboot :)
