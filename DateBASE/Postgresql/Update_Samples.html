Примение соединенияе

Обновление данных в табдице  "numbers" получая данные из таблицы "route_e164":

UPDATE "numbers" SET "gateway" = "route_e164"."gw_uuid",
                   "status" = "route_e164"."status"
 FROM "route_e164" WHERE "route_e164"."number" = "numbers"."num";
