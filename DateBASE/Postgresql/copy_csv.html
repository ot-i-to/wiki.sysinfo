Индикация прогресса выполнения команды COPY :
# pv имя.csv файла | sudo -u postgres psql -d имя базыданных -c "copy таблица(колонка-1, колонка-2, колонка-3, колонка-N) from stdin delimiter ';' header null ''"

Результат будет такой:

1.42GiB 0:11:42 [2.06MiB/s] [=====================================================================>] 100%
