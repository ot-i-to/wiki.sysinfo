SIP Error Messages 

Тут показаны наиболее распространенные ошибки, которые можно преобразовать в сообщения об ошибках Q.931 или DSS1, они соответствуют следующим типам:
4xx — Сбои метода.
5xx — Сбои сервера.
6xx — Глобальные неудачи.

Ошибки SIP и ошибки DSS1 / Q.931 сопоставляются, как показано в таблице.
