Ошибка Call to undefined function split()

Ошибка появляется из-за того что в php 7.0.0 метод split() удален.
А решение ошибки очень простое, нужно просто заменить его на метод explode().

Было:

$domainsName = split( '\r\n', $originList['domain'] );
$domainsName = split( '\r\n', $originList['domain'] );

Стало:

$domainsName = explode( '\r\n', $originList['domain'] );
$domainsName = explode( '\r\n', $originList['domain'] );
